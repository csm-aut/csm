Skip the hardware audit for fans and power modules?

     

Minimal hardware requirements for migration:
   1. All RSP's and RP's are supported in ASR9K-X64 and are in IOS XR RUN state
   2. All FC's are supported in ASR9K-X64 and are in OK state.
   3. All MPA's are supported in ASR9K-X64 and are in OK state.
   4. All supported LC's are in IOS XR RUN state.
By default, on top of the minimal hardware requirements, the hardware audit checks that all FAN's and
PEM's are supported in ASR9K-X64 and are in READY state.

Operational state for RSP, RP and LC: IOS XR RUN
Operational state for FC: OK
Operational state for FAN and PEM: READY
Operational state for MPA: OK

The list of the hardware supported in ASR9K-X64 is available in Help > ASR9K to ASR9K-X64 Migration.
